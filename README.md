# ai_vision Image Classifier with Transfer Learning

A high-performance image classifier for CIFAR-10 dataset using ResNet18 with transfer learning from ImageNet.

## Features

- ✅ Transfer learning with pretrained ResNet18
- ✅ Advanced data augmentation (random flip, rotation, color jitter, affine transforms)
- ✅ Training/validation split (90/10)
- ✅ Learning rate scheduling (Cosine Annealing)
- ✅ Comprehensive evaluation metrics
- ✅ Training curves visualization
- ✅ Confusion matrix
- ✅ Sample predictions visualization
- ✅ Easy inference script

## Results

- **Test Accuracy**: ~90-92% (depending on random seed)
- **Training Time**: ~30 minutes on GPU (30 epochs)
- **Model Size**: ~44 MB
